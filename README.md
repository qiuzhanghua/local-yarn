# local-yarn
process yarn.lock


## Zip
on your unix machine, under nodejs project home, run
```bash
./gen-yarn-zip <name>
```

## Setup
```bash
yarn global add https://github.com/qiuzhanghua/local-yarn.git#0.6.1
```

## Install Zip
```bash
local-yarn install <name>.yarn.lock.zip
```

## update local yarn.lock
```bash
local-yarn get meta
```
